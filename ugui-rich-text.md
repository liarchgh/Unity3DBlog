# UGUI Rich Text
## b
效果：加粗
例子：`<b>TestTest</b>`
## i
效果：斜体
例子：`<i>TestTest</i>`
## size
效果：大小
例子：`<size=40>TestTest</size>`
## color
效果：颜色
例子：`<color=#00ffffff>TestTest</color>`
## material
效果：材质
例子：`<material=1>TestTest</material>`
